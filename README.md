Summary of learning!VS2013常用代码相关快捷键
VS2013常用代码相关快捷键
Shift+Alt+Enter 全屏显示代码区
Ctrl+K+F 代码自动对齐
Ctrl+J 调用智能提示

Alt+→ 调用智能提示



Ctrl+M+M 显示/隐藏代码段



Ctrl+K+C 注释

Ctrl+K+U 取消注释



Ctrl+C 复制光标所在整行

Ctrl+X 剪切光标所在整行

Ctrl+L 删除光标所在整行



Ctrl+Z 撤销操作

Ctrl+Y 反撤销



Ctrl+Enter 当前行上一行插入空行

Ctrl+Shift+Enter  当前行下一行插入空行


 

F12 跳转到定义


 

Ctrl+Shift+空格 高亮参数在函数内使用位置


 

Ctrl+Shift+B 生成解决方案


Home 定位至当前行行首

End  定位至当前行行尾


 

Shift+Home 选中行首到光标处代码

Shift+End  选中光标处到行尾代码


 
 Shift+Alt+方向键 框式选择
 


 

F9 切换断点

F5 启动调试

Shift+F5 停止调试

Ctrl+Shift+F5 重新调试

Ctrl+F5 开始执行（不调试）

F11 逐语句调试

F10 逐过程调试

Shift+Alt+方向键    框式选择
